# Typography Rules

Visual hierarchy and font specifications.

## 1. Report Title
- Component: `<ReportTitle>`
- Style: Serif (`font-serif`)
- Size: `text-xl` (20px)
- Weight: `font-bold`
- Decoration: `decoration-double`
- Color: `text-blue-900`

## 2. Section Headers
- Size: `text-[15px]` or `text-base` (16px) - **larger than body**
- Weight: `font-bold`
- Case: Uppercase
- Color: `text-blue-900`
- Decoration: `border-b`

## 3. Subsection Headers
- Size: `text-sm` (14px) or `text-base`
- Weight: `font-bold`
- Color: `text-gray-700` or black

## 4. Body Text / Inputs
- Size: `text-sm` (14px) - **NOT `text-xs`**
- Weight: Normal or Semibold

## 5. Tables
- Size: `text-xs` (12px)
- Headings: `font-bold`, `bg-gray-100`

## 6. Metadata / Footers
- General: `text-[10px]`
- Remarks: `text-[11px]`
- Color: `text-gray-500`

## 7. References
- Size: `text-xs` (12px)
- Color: `text-gray-600`
