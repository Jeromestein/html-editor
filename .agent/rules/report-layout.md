# Report Layout

FCE report structure and section requirements.

## 1. Global Sections (Start of Report)

### 1. U.S. EQUIVALENCE SUMMARY
- Lists all credentials (labeled Credential #1, #2, etc.)
- Contains: Equivalency, U.S. Credits, U.S. GPA

### 2. DOCUMENTS
- Single global list of submitted documents

---

## 2. Per-Credential Sections

Repeated for each credential:

### 3. CREDENTIAL DETAILS
- Header: "Credential Details: Credential #X"
- Institution, Country, Year, Major, etc.
- **Institution Name Rule**: English name. Append native name in parentheses if needed.

### 4. COURSE-BY-COURSE ANALYSIS
- Header: "Course-by-Course Analysis: Credential #X"
- Table: courses, grades, credits, levels
- Year column: `w-20` for "YYYY-YYYY" format
- Lvl column: hidden by default
- "+ Add Course" button in table footer

### TOTALS
- After course table
- Shows Total Credits and GPA for that credential

### 5. GRADE CONVERSION
- Header: "Grade Conversion: Credential #X"
- Grade scale table
- **Pagination**: Fill courses first, only break if needed

---

## 3. Footer & Closing Sections

### REFERENCES
- Standard reference databases (AACRAO EDGE)
- No global grade scale table

### EVALUATION NOTES
- Editable text area

### REMARKS / SIGNATURES
- Disclaimer and signature lines
- No global totals
